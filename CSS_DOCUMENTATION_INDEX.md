## üì± Responsive Breakpoints

Know where to put your media queries:

```css
/* Mobile (default) */
.my-element { font-size: 14px; }

/* Tablet (768px+) */
@media (min-width: 768px) {
  .my-element { font-size: 16px; }
}

/* Desktop (1024px+) */
@media (min-width: 1024px) {
  .my-element { font-size: 18px; }
}
```

**See:** [CSS_QUICK_REFERENCE.md - Responsive Design](CSS_QUICK_REFERENCE.md)

## üí° Pro Tips

1. **Use CSS Variables**
   ```css
   ‚úÖ color: var(--color-primary);
   ‚ùå color: #2563eb;
   ```

2. **Use Utility Classes**
   ```css
   ‚úÖ <div class="flex-center p-24">
   ‚ùå <div style="display: flex; padding: 24px;">
   ```

3. **Check DevTools**
   - F12 ‚Üí Elements ‚Üí Styles
   - See which CSS file applies each rule
   - Click filename to jump to CSS

4. **Test Responsive**
   - F12 ‚Üí Responsive mode
   - Test 375px (mobile), 768px (tablet), 1024px (desktop)

5. **Keep It DRY**
   - Don't repeat CSS
   - Create utility classes instead
   - Use variables for reusable values
